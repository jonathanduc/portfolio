Challenge Mobilité

Projet dans le cadre d’un projet Erasmus + 3CS “Cycling, Campus & City” en partenariat avec la start-up Muv. Objectifs côté MIASHS : traitement et analyse de données de l'application Muv, développement d'une application pour visualiser les données. Utilisation de données brutes (~500 000 lignes), analyses descriptives, exploratoires et clustering avec K-means.

Méthodologie

Conception de l’application Cyclers :
Récolte de données : Utilisation de l'application MUV pour collecter des données brutes (~500 000 lignes).
Analyses descriptives et exploratoires : Études préliminaires pour identifier les données aberrantes et comprendre les tendances générales.
Cartographie : Visualisation des trajets parcourus par les utilisateurs sur une carte interactive.
Clustering : Utilisation de l’algorithme K-means pour identifier des groupes de profils utilisateurs.
Communication : Promotion du challenge pour encourager les utilisateurs à participer et à utiliser l'application.

Technologies Utilisées

  • Python
  • R
  • Leaflet
  • Shiny
  • K-means
