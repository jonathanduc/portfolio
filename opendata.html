Open Data Dashboard

Projet en binôme — M2 (Open Data & Web des données)

Développement d’un pipeline d’analyse et d’un tableau de bord interactif à partir de jeux de données publics (data.gouv.fr). Le projet inclut la collecte, le nettoyage, une mini-API pour exposer des indicateurs, et des visualisations Plotly.

Objectifs

  • Consolider des jeux de données publics hétérogènes (CSV, API) dans un schéma cohérent.
  • Exposer des indicateurs via une API Flask simple et documentée.
  • Offrir un dashboard interactif pour l’exploration.

Méthodologie

  • Ingestion : extraction depuis data.gouv.fr (CSV / endpoints), contrôle qualité (types, NA, doublons).
  • Préparation : normalisation, enrichissement (features dérivées), index temporel.
  • Service : endpoints Flask pour KPIs (JSON) + pages HTML légères.
  • Visualisation : graphiques Plotly (séries, cartes, bar charts).

Résultats (exemples)

  • Latence API < 150 ms sur endpoints principaux (local dev).
  • 3–5 indicateurs exposés (moyenne mobile, agrégations par région, top-N).
  • Dashboard multi-onglets, filtres par période & zone.

Stack & Outils

  • Python, Flask, Pandas
  • Plotly (visualisation)
  • Requests, JSON
